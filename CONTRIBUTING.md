# Contributing

## Goals

USDX Layer aims to provide a deterministic, verifiable execution and routing abstraction for USD1 activity.

## Workflow

1. Fork the repository
2. Create a feature branch
3. Add tests for behavior and invariants
4. Run npm run lint, npm run typecheck, npm test
5. Submit a pull request with a clear description

## Style

- Prefer pure functions
- Use explicit types
- Avoid hidden state
- Validate inputs at module boundaries
